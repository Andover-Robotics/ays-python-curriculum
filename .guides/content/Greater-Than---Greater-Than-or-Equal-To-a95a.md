## Greater Than

The `>` operator is used to check if one value is greater than another value.

```python
a = 9
b = 17
print(a > b)
```

## Greater Than or Equal To

The `>=` operator is used to check if one value is greater than or equal to another value.
```python
a = 9
b = 17
print(a >= b)
```
